# Asset Layout

이 폴더는 웹 플레이 버전에서 사용하는 외부 아트 에셋을 관리합니다.

## 구조

- `sprites/player/`: 플레이어 렌더용 스프라이트
- `sprites/monsters/`: 몬스터 아키타입별 스프라이트 (현재 human_* 중심 매핑)
- `sprites/tiles/`: 바닥/플랫폼 텍스처 오버레이
- `sprites/ui/`: HUD 아이콘
- `docs/`: 라이선스/출처 문서
- `sources/`: 원본 패키지(zip/압축해제 원본)
