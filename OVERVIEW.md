## 1. 개요 및 핵심 루프 (Overview & Game Loop)

### 1.1. 하이 컨셉

**"퇴근을 위해 회사라는 던전을 탈출하라!"**

지하 주차장(B6)부터 대표이사실(9F)까지 올라가며 업무 스트레스 괴물들을 처치하는 2D 로그라이트 액션.

### 1.2. 게임 진행

1. `B6`에서 시작해 층을 클리어하며 상층으로 이동
2. 각 층에서 일반 몬스터/보스/임원을 상대
3. `9F` 클리어 후 결과 및 랭킹 기록 저장
4. 클리어 후 재도전 방식 선택
: 상태 초기화 재시작 또는 현재 상태 유지 재도전

## 2. 현재 구현 기술/데이터 기준

- 실제 실행/배포: `playable-web/*`
- 저장: LocalStorage
  - `rescaperMeta`, `rescaperSave`, `rescaperSettings`
- 렌더링: HTML5 Canvas 2D (**RenderSystem 모듈화**)
- 아키텍처: **ES Modules (ESM) 기반 모듈화 구조**
  - **Core 엔진 (`game.js`):** 메인 루프 및 전역 상태(State) 관리
  - **데이터 설정 (`systems/data-config.js`):** 층 구성, 무기, 스탯 등 정적 데이터 분리
  - **기능 시스템 (`systems/*`):** 렌더링, 전투, 오디오, FX, 저장, 자산 관리 등 기능별 독립 모듈

랭킹 정렬 규칙:
1. 총 클리어 횟수 높은 순
2. 동률 시 클리어 시간 짧은 순

---

## 3. 층/전투 시스템 요약

- 층 구성: `B6 ~ 9F` (B1은 Safe Zone)
- 공통 시스템:
  - 무기/어픽스
  - 랜덤 수치 스킬 선택
  - 전투 스타일
  - 콤보/오버드라이브
  - 적 변이(mutator)
- 보스전:
  - 보스/임원 페이즈(1~3) 전환
  - 페이즈별 스킬 쿨타임/압박 패턴 강화
- 층 콘텐츠:
  - 상점(B1)
  - 층별 특수 이벤트 룸(E 상호작용)

---

## 4. UI/HUD 요약

- 상단: HP/XP, 층/목표/타임어택, 보스 상태
- 하단: 소비 슬롯, 골드, 콤보/HEAT/OD
- 추가 HUD:
  - 무기 고유 패시브 패널
  - 발동 상태 표시
- 패널 토글:
  - `H`로 조작법/로그/상태 패널 숨김/표시

---

## 5. 조작 키

- 이동: `← / →`
- 점프: `↑`
- 공격: `Space`
- 대시: `Shift`
- 상호작용/상점/이벤트: `E`
- 소비 슬롯: `Q`
- 스킬 선택: `1 / 2 / 3`
- 재시작(사망): `R`
- 정보 패널 토글: `H`
- 이스터에그 층 이동: `+ / -`
- 볼륨: `Alt + + / Alt + -`
- 음소거: `M`
- 이펙트 간소화: `V`
- 일시정지: `P`
